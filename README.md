# Barcode Scanner Portal

Mobile-first barcode scanning interface for inventory management and product tracking.

## Features

- Mobile-optimized barcode scanning
- Product management
- Real-time inventory tracking
- Session history

## Live Demo

[https://www.greyfielddata.com/scanner](https://www.greyfielddata.com/scanner)

## Tech Stack

- Vanilla HTML/CSS/JavaScript
- Mobile-first responsive design
- LocalStorage for data persistence

## Usage

Open `index.html` in a web browser or deploy to any static hosting service.
